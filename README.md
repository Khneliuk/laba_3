# Лабораторна робота 3
Класи та об'єкти. Конструктори та деструктори. Модифікатори. Інкапсуляція

Завдання 1 (варіант 30):

Працівник. Компанія зберігає дані про працівників: унікальний ID, ім’я та зарплату. ID змінювати не можна. Компанія веде підрахунок створених об’єктів. Створіть клас Employee з прихованими полями (private). ID має бути незмінним (readonly). Для підрахунку використайте спільне поле (static). Передбачте створення працівника з початковою зарплатою і без неї (зарплата = 0). Реалізуйте метод підвищення зарплати (відсотком) та метод обчислення річного доходу. Зарплату отримувати через властивість тільки для читання (property get). Додайте статичний метод підрахунку (static method) і перевизначте ToString() (override). У Main() протестуйте логіку.

Завдання 2:

Створити у попередньому завданні два методи з використанням серіалізації та десеріалізації JSON. 
Метод 1. Зберігає створений об’єкт класу з Завдання 1 у JSON файл 
Метод 2. Відкриває JSON файл з даними та створює об’єкт класу з цими даними для виконання Завдання 1.
